# Security Policy

## Reporting a Vulnerability
If you believe you have found a security vulnerability, please report it privately.

Preferred options:
- Email: chanproseth.kan@gmail.com

Please include:
- A description of the issue and potential impact
- Steps to reproduce
- Any proof-of-concept code or logs

We will acknowledge receipt within 3 business days and will provide a timeline for remediation after triage.
